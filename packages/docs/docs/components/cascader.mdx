# Cascader

Cascader is a dropdown menu for selecting a value from a list of options that can be nested.

### Import

```jsx
import { Cascader } from "@sparrowengg/twigs-react";
```

type CascaderOption = {
  label: string;
  value: string;
  options?: CascaderOption[];
